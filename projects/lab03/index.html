What I Learned

  1. I learned the distinction between the bold element and the strong element. The bold element is primarily used to grab the reader’s attention, while the strong element conveys importance. The purpose of the strong element becomes especially clear when used with a screen reader for the visually impaired.
  2. I also learned about the multiple ways to style HTML documents using CSS (Cascading Style Sheets). For example, this is a big green sentence and this is a small red one.
  3. Another thing I learned about was access to symbols that aren't on the keyboard. The Konami Code is the most famous cheat code in video game history. It goes as follows: ↑ ↑ ↓ ↓ ← → ← → B A START.

Reflections

Lab02 was my reintroduction to HTML, and I found the syntax to be straightforward and easy to understand. At first, I felt overwhelmed by the number of elements available, but I soon realized that mastering the essential ones was enough. Working through the W3Schools tutorial helped reinforce my understanding and gave me a solid grasp of the fundamentals. This experience made me more confident in using HTML and prepared me for more advanced concepts.

After completing the previous assignment, Lab03 was easier to understand. The biggest challenge was deciding how to style the HTML page. With so many possibilities, the design process was both exciting and somewhat time-consuming. At first, using a stylesheet felt overwhelming, but it ended up making the styling process much more efficient.
